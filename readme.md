# ruled-app

Mini app to create css debug grids.
This was built around [`ruled`](https://github.com/jxnblk/ruled) by [@jxnblk](https://twitter.com/jxnblk).

## Installation

```bash
git clone https://github.com/johnotander/ruled-app
cd ruled-app
npm i
npm run dev
open http://localhost:3000
```
